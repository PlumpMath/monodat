# monodat

A little of monodis, a little of monodat

## Status

Not finished/in progress

## Usage

```
$ node monodat.js Assembly.dll.cil
```

Where `Assembly.dll.cil` is a dump from `monodis`

Eventually we will have

```
$ monodis Assembly.dll | monodat
```
